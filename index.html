Road to Dunk

  1. On any 2 free days, we will focus on weight lifting. Begin with isometric warm-ups to prepare your muscles for the workout.
  2. On basketball days, perform the Isaiah warm-up before hitting the court.

Weight Lifting Program

  1. Start with isometric warm-ups and bodyweight squats.
  2. Deadlift for 3 sets of 6 reps.
  3. Weighted jumps for 3 sets of 7 reps.
  4. Choose either Power Clean or High Pull for 4 sets of 5 reps.
  5. Half weighted squats for 3 sets of 7 reps.
  6. Romanian Deadlift (RDL) for 2 sets of 6 reps.

Warm Up

Before we start, run a full court 6.

  1. QTR Depth Lunge - First set for 45 sec each leg.
  2. Between sets, perform dynamic flexibility exercises:
    • 10× Hip Hings
    • 10× Body Weight Squats
    • 10× Body Weight Lunges (5 each leg)
    • 20× Arm Circles
    • 20× Goku Side Lunge (10 each side)
  3. QTR Depth Lunge - Second set for 45 sec each leg.
  4. Continue dynamic flexibility after the second set:
    • 10× Forward Leg Swing on each leg
    • 10× Side Leg Swing on each leg
    • 10× Butt Stretch (5 on each leg)
    • 10× Hurdle Mobility on each leg
  5. Jump Warm Up - Start with baby jumps and gradually increase power.
